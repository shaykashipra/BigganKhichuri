# Bigganখিচুড়ি - AI-Powered Science Learning Platform

**An inclusive, interactive science education platform for rural students with AI chatbots, voice commands, and gamified learning.**



## 📋 Overview

Bigganখিচুড়ি is a web-based educational platform designed to revolutionize science education in rural Bangladesh. The platform combines **AI-powered chatbots**, **voice-to-text technology**, **interactive simulations**, and **gamified quizzes** to create an engaging, accessible learning experience for secondary school students.

### 🎯 Why This Project?

- **Rural Access Gap**: Limited access to quality science education in rural Bangladesh
- **Learning Challenges**: Complex science concepts need interactive, visual explanations
- **Accessibility Issues**: Students with disabilities lack inclusive learning tools
- **Engagement Problem**: Traditional learning methods fail to motivate students

---

## ✨ Key Features

### 1. **AI-Powered Chatbot**
- Personalized learning support using GPT-4
- Adaptive learning paths based on student progress
- Real-time doubt resolution

### 2. **Voice & Accessibility**
- Voice-to-text input for hands-free learning
- Text-to-speech for content delivery
- Support for students with visual, auditory, and motor impairments

### 3. **Interactive Learning**
- Physics, Chemistry, and Biology simulations
- Real-time 3D visualizations of complex concepts
- Hands-on experiments in a safe digital environment

### 4. **Gamified Quizzes**
- Progress tracking and performance analytics
- Leaderboards and achievement badges
- Customized assessments generated by AI

### 5. **Collaboration Hub**
- Student-to-student idea sharing
- Community learning features
- Peer support networks



---

## 🏗️ Technical Architecture

```
Frontend (React.js)
    ↓
Backend (Python Flask)
    ↓
Database & Auth (Supabase)
    ↓
AI/LLM (OpenAI GPT-4 API)
```

### Tech Stack

| Component | Technology |
|-----------|-----------|
| **Frontend** | React.js (Responsive UI/UX) |
| **Backend** | Python Flask (REST APIs) |
| **Database** | Supabase (Real-time, PostgreSQL) |
| **Authentication** | Supabase Auth |
| **AI/LLM** | OpenAI GPT-4 API |
| **Voice Processing** | Web Speech API |
| **Hosting** | Cloud-ready (Scalable) |
